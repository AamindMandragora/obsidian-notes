Verbs that end in "-cer" and "-ger" have a different conjugation when used with "nous" to preserve the soft "c" and "g" sound before the "ons". Specifically, "c" becomes "ç" and "g" becomes "ge".

| French    | English             |
| --------- | ------------------- |
| commencer | to start            |
| corriger  | to correct          |
| manger    | to eat              |
| partager  | to share            |
| rédiger   | to write / compose  |
| exiger    | to demand / require |
| nager     | to swim             |
| ranger    | to tidy / arrange   |
| songer    | to dream / reflect  |
| voyager   | to travel / journey |

Verbs that end in "-yer" have a different conjugation with "je", "tu", "il / elle", and "ils / elles". For those, the "y" becomes an "i" before silent "e"s.

| French      | English           |
| ----------- | ----------------- |
| (s')ennuyer | to bore (oneself) |
| envoyer     | to send           |
| nettoyer    | to clean          |
| payer       | to pay            |

Verbs with an "e" in the second last syllable also have different conjugations for the same subjects as above. For those, the "e" becomes an "é".

| French  | English                   |
| ------- | ------------------------- |
| acheter | to buy                    |
| amener  | to bring somebody (along) |
| emmener | to take somebody (along)  |
| lever   | to lift / raise           |
| mener   | to take / lead            |
| peser   | to weigh                  |

Most verbs that end in "-eler" and "-eter" also have different conjugations for those subjects. For those, double the "l" or "t".

| French      | English           |
| ----------- | ----------------- |
| (s')appeler | to call (oneself) |
| jeter       | to throw          |

Verbs that have an é in the second last syllable also have different conjugations for those subjects. For those, the "é" becomes a "è".

| French     | English                     |
| ---------- | --------------------------- |
| considérer | to consider                 |
| espérer    | to hope                     |
| exagérer   | to exaggerate               |
| préférer   | to prefer                   |
| régler     | to regulate / settle        |
| répéter    | to repeat                   |
| sécher     | to dry or to skip (a class) |

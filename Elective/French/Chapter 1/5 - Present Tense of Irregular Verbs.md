Some of the most common verbs are irregular, like être, avoir, aller, dire, and faire.

| être        | avoir      | aller       | dire        | faire          |
| ----------- | ---------- | ----------- | ----------- | -------------- |
| 'to be'     | 'to have'  | 'to go'     | 'to say'    | 'to do / make' |
| je suis     | j'ai       | je vais     | je dis      | je fais        |
| tu es       | tu as      | tu vas      | tu dis      | tu fais        |
| il est      | il a       | il va       | il dit      | il fait        |
| nous sommes | nous avons | nous allons | nous disons | nous faisons   |
| vous êtes   | vous avez  | vous allez  | vous dites  | vous faites    |
| ils sont    | ils ont    | ils vont    | ils disent  | ils font       |

Some "-ir" verbs drop the last letter in the stem and follow the following pattern: "-s", "-s", "-t", "-ons", "-ez", "-ent".

| French      | English                   |
| ----------- | ------------------------- |
| dormir      | to sleep                  |
| partir      | to depart (to somewhere)  |
| (se) sentir | to feel (internally)      |
| servir      | to serve                  |
| sortir      | to leave (from somewhere) |
| mentir      | to lie                    |

Some verbs that end in "-dre" drop the "d" from the stem in their plural forms.

| French     | English       |
| ---------- | ------------- |
| apprendre  | to learn      |
| comprendre | to understand |
| prendre    | to take       |
| surprendre | to surprise   |

Some verbs can be grouped simply because they share the same endings: "-s", "-s", "-t", "-ons", "-ez", "-ent". They all have two different stems in the singular and plural (usually the addition of an extra consonant).

| French    | English                 | Singular Stem | Plural Stem |
| --------- | ----------------------- | ------------- | ----------- |
| mettre    | to put                  | met           | mett        |
| conduire  | to conduct              | condui        | conduis     |
| traduire  | to translate            | tradui        | traduis     |
| connaître | to know (noun)          | connai        | connaiss    |
| craindre  | to fear                 | crain         | craign      |
| décrire   | to describe             | décri         | décriv      |
| écrire    | to write                | écri          | écriv       |
| lire      | to read                 | li            | lis         |
| savoir    | to know (verb / clause) | sai           | sav         |
| suivre    | to follow               | sui           | suiv        |
| vivre     | to live (like)          | vi            | viv         |
For the verbs croire and voir, they use the stem "croy" / "voy" for "nous" and "vous", and "croi" / "voi" for the rest of the subjects.

Some verbs have three different stems for "je" / "tu" / "il", "nous" / "vous", and "ils" / "elles", and there's really no good pattern for them.

| French   | English       | je / tu / il | nous / vous | ils / elles |
| -------- | ------------- | ------------ | ----------- | ----------- |
| tenir    | to hold       | tien         | ten         | tienn       |
| venir    | to go         | vien         | ven         | vienn       |
| boire    | to drink      | boi          | buv         | boiv        |
| devoir   | to have to    | doi          | dev         | doiv        |
| recevoir | to recieve    | reçois       | recev       | reçoiv      |
| pouvoir  | to be able to | peu          | pouv        | peuv        |
| vouloir  | to want       | veu          | voul        | veul        |

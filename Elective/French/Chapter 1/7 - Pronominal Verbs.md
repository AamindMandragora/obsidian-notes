Pronominal verbs have a pronoun + verb and otherwise work the same. For negations, the reflexive pronoun goes inside the "ne... pas" Questions are usually of the form "est-ce que...", but for inversion the subject succeeds the verb and the reflexive pronoun precedes it. In the infinitive, the reflexive pronoun agrees with the subject.

There are three types of pronominal verbs. The first is reflexive, where the subject does the action to themselves.

| French       | English         |
| ------------ | --------------- |
| s'assesoir   | to sit down     |
| s'appeler    | to be called    |
| s'arrêter    | to stop         |
| se baigner   | to bathe        |
| se brosser   | to brush        |
| se coucher   | to go to bed    |
| se doucher   | to shower       |
| s'endormir   | to fall asleep  |
| s'habiller   | to dress        |
| se laver     | to wash         |
| se lever     | to get up       |
| se maquiller | to apply makeup |
| se peigner   | to comb hair    |
| se promener  | to take a walk  |
| se raser     | to shave        |
| se réveiller | to wake         |

The second type is reciprocal, where the subject does the action to or with another.

| French        | English        |
| ------------- | -------------- |
| s'aimer       | to love        |
| se comprendre | to understand  |
| s'écrire      | to write       |
| s'embrasser   | to hug or kiss |
| se quitter    | to leave       |
| se recontrer  | to encounter   |
| se retrouver  | to meet up     |
| se voir       | to see         |
| se détester   | to hate        |
| se disputer   | to argue       |
| se parler     | to talk        |
| se regarder   | to look        |
| se téléphoner | to telephone   |

The third type is idiomatic, where the reflexive pronoun changes the meaning.

| French              | English             |
| ------------------- | ------------------- |
| s'en aller          | to leave / go away  |
| s'amuser            | to have fun         |
| se débrouiller      | to manage / get by  |
| se demander         | to wonder           |
| se dépêcher         | to hurry            |
| s'ennuyer           | to be / get bored   |
| s'entendre          | to get along        |
| s'inquieter         | to worry            |
| se mettre á         | to begin / start    |
| se rendre compte de | to realize          |
| se tromper (de)     | to be wrong (about) |
| se fâcher           | to get angry        |
| se marier           | to get married      |
| se passer           | to happen           |
| se reposer          | to rest             |
| se sentir           | to feel (within)    |
| se souvenir de      | to remember         |
| se taire            | to be silent        |
| se trouver          | to be located       |

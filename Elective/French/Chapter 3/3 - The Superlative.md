In French, we can express the idea of the most or the least between three or more groups in terms of adjectives, adverbs, verbs, and nouns using the superlative (very similar to the comparative).

| Part of Speech | Supériorité                        | Infériorite                         |
| -------------- | ---------------------------------- | ----------------------------------- |
| adjectives     | le / la / les plus [adjectif.ve.s] | le / la / les moins [adjectif.ve.s] |
| adverbs        | le plus [adverbe]                  | le moins [adverbe]                  |
| verbs          | [verbe] le plus                    | [verbe] le moins                    |
| nouns          | le plus de [nom]                   | le moins de [nom]                   |

Just like in the comparative, "bon / mauvais / bien" have special superlative forms for "the most".

| Part of Speech | French      | Superiorité                |
| -------------- | ----------- | -------------------------- |
| adjective      | bon.ne.s    | le / la / les meilleur.e.s |
|                | mauvais.e.s | le / la / les pire.s       |
| adverbe        | bien        | le mieux                   

Adjectives stay in their normal place relative to the noun, which the definite article agrees with. For adverbs, verbs, and nouns, the definite article is always "le". To indicate "in" or "of" with the superlative, use the definite article prepended with "de" ("du / de la / des").

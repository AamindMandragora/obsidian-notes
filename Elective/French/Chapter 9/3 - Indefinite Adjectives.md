To express the indefinite idea of quality or quantity, we can use indefinite adjectives, most of which still agree in number and gender with the noun they modify. Some can even be used as indefinite pronouns.

| French        | English             |
| ------------- | ------------------- |
| ne... aucun.e | not one / not a one |
| certain.e(s)  | certain / some      |
| chaque        | each / every        |
| plusieurs     | several             |
| quelque(s)    | a few               |
| autre(s)      | other               |

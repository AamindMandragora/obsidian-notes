Infinitives are unconjugated verbs and can be used as impersonal orders, as a subject (uncommon), or as the complement of a verb. Verbs related to movement, inclination, opinion, or perception are followed directly by an infinitive.

| Class       | French   | English       |
| ----------- | -------- | ------------- |
| Movement    | aller    | to go         |
|             | partir   | to leave      |
|             | rentrer  | to re-enter   |
|             | revenir  | to return     |
|             | venir    | to come       |
| Inclination | aimer    | to like       |
|             | désirer  | to desire     |
|             | détester | to detest     |
|             | pouvoir  | to be able    |
|             | préférer | to prefer     |
| Opinion     | compter  | to count      |
|             | croire   | to believe    |
|             | devoir   | to be obliged |
|             | espérer  | to hope       |
|             | falloir  | to need       |
|             | penser   | to think      |
|             | savoir   | to know       |
| Perception  | écouter  | to listen     |
|             | entendre | to hear       |
|             | regarder | to watch      |
|             | voir     | to see        |

Some verbs must be followed by the preposition "à" before an infinitive.

| French     | English           |
| ---------- | ----------------- |
| aider      | to help           |
| s'amuser   | to amuse oneself  |
| apprendre  | to learn          |
| commencer  | to start          |
| encourager | to encourage      |
| s'habituer | to get accustomed |
| hésiter    | to hesitate       |
| inviter    | to invite         |
| réussir    | to succeed        |

Other verbs must be followed by the preposition "de" before an infinitive.

| French                  | English                  |
| ----------------------- | ------------------------ |
| s'arrêter               | to stop oneself          |
| avoir besoin            | to have need             |
| s'excuser               | to excuse oneself        |
| conseiller              | to advice                |
| décider                 | to decide                |
| essayer                 | to try                   |
| être content.e / ravi.e | to be content / thrilled |
| finir                   | to finish                |
| promettre               | to promise               |
| mériter                 | to merit                 |
| oublier                 | to forget                |
| se presser              | to hurry                 |
| refuser                 | to refuse                |
| avoir envie             | to have desire           |
| rêver                   | to dream                 |
| se souvenir             | to remember              |

There's no rule or pattern to distinguish from the latter two types of verbs, so they must be memorized.
